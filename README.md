
# Personal-Spending-Analysis
Analysis of bank data from using Excel to identify trends and provide insights with spending behavior

## Objective
The goal of this project is to understand where money is being spent and identify opportunites to reduce unnecessary expenses

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers

## Key Features
- Interactive dashboard
- Spending by Category analysis
- Spending over time
- Summary metrics (toals, average, highest expense)

## Insights
- The highest Spending category is Groceries
- Total Spending decreased by 25.4% from February to March
- Reducing grocery spending by 20% would lower its share of total spending from 47% to approximately 42%.
